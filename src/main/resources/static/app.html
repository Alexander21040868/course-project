📖 Создать урок

Правка урока: id и «Подставить в форму», затем «Сохранить». «Новый урок» — сброс.

Эти задачи будут перенесены в текущий урок при сохранении. Поле необязательное.

🔗 Прикрепить или открепить задачи

Введите номер подземелья и id задач. Прикрепление не отнимает задачу у других уроков — каждая задача может быть в нескольких подземельях.

⚔ Создать задачу

Правка задачи: id и «Подставить в форму». «Новая задача» — сброс.

Для условия задачи: что подаётся на вход программе и что должно получиться на выходе. Строка сохраняется, если заполнен блок «Выход». Эти же пары участвуют в автопроверке решения.

Только при создании новой задачи. К уже существующим челлендж не подключается.

📚 Подгруппы

Делите своих учеников на подгруппы (классы, потоки). Учеников можно перемещать между ними.

👥 Моя группа

Здесь видны только ваши ученики. Можно фильтровать по подгруппе и искать по имени.

➕ Пригласить ученика в группу

Ученик получит уведомление и сможет принять или отклонить приглашение.

🏟 Создать челлендж

Задачи к соревнованию не цепляются списком id: добавьте их вручную в «Контент» — новая задача и выбор челленджа; после старта соревнования они попадают в общий каталог.
